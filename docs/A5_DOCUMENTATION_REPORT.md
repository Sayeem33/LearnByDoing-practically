## Project Title

**SimuLab: Virtual Lab and Concept Learning Platform**

---

## Project Summary

SimuLab is a web-based educational platform that helps students learn difficult concepts through:

- interactive virtual labs
- guided tutorials
- progress tracking
- report generation
- validation and evidence summaries
- instructor review
- assignment workflow
- concept authoring for new modules

The system currently supports:

- **Physics** modules
- **Chemistry** modules
- **Math concept visualizations**

It is designed mainly for high school and early undergraduate learners, while giving teachers and admins tools to manage, review, assign, and expand learning content.

---

## Main Implemented Capabilities

- Student registration and login
- Role-based access for student, teacher, and admin
- Lab workspace with save, complete, submit flow
- Tutorial pages with chapter tracking
- Validation and evidence generation for experiment runs
- Instructor review queue for submitted labs
- Assignment workflow for classes and students
- MongoDB-backed progress tracking
- Experiment authoring studio for new concept definitions
- Demo showcase dashboard for presentation and viva support
